Value

Showing negative, positive and nochange values. Mainly used in financial applications.

Positive: 100.00 
Positive: <span class="financial__value financial--positive">100.00</span>
No Change: 00.00 
No Change: <span class="financial__value financial--nochange">00.00</span>
Negative: -80.95 
Negative: <span class="financial__value financial--negative">-80.95</span>
Normal: -80.95 
Normal: <span class="financial__value">-80.95</span>

Financial Colors

There are three 3 colors for financial values: positive, negative, nochange, normal (default) . Simply apply the type modifier class for the desired color.

Value Modifier Class
100.00 financial--positive
00.00 financial--nochange
-80.95 financial--negative
80.95 financial__value
